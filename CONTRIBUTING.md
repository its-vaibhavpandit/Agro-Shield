# Contributing to AgroShield AI

Thank you for your interest in contributing to AgroShield AI! We welcome contributions from the community.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/yourusername/agroshield-ai.git`
3. Create a new branch: `git checkout -b feature/your-feature-name`
4. Make your changes
5. Commit your changes: `git commit -am 'Add your feature'`
6. Push to your fork: `git push origin feature/your-feature-name`
7. Create a Pull Request

## Code Style

- Follow Java naming conventions (camelCase for variables/methods, PascalCase for classes)
- Keep methods focused on a single responsibility
- Add comments for complex logic
- Ensure all code compiles without warnings

## Testing

- Test your changes thoroughly before submitting
- Recompile using: `python compile.py`
- Run the application: `java -cp bin com.agroshield.Main`

## Reporting Issues

If you find a bug, please create an issue with:
- Clear description of the problem
- Steps to reproduce
- Expected vs actual behavior
- Screenshots if applicable
